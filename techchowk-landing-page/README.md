## Challenge Description

**Goal:**
Your task is to build a modern, responsive landing page for our community, TechChowk. This page should introduce what we do, and encourage new developers to join us.

**Requirements:**

1. Navigation Bar: A sticky or fixed header with links (Home, Challenges, About, Join).
2. Hero Section: A bold headline (e.g., "Where Nepal's Tech Minds Meet"), a short sub-headline, and a "Join Discord" Call-to-Action (CTA) button.
3. Features/About Section: Use a Grid or Flexbox layout to show 3 cards (e.g., "Peer Learning," "Weekly Challenges," "Networking").
4. Responsive Design: The site must look good on both a desktop and mobile screen.
5. Clean Code: Use semantic HTML5 tags.
6. (Optional) Dark/Light Mode: A toggle to switch themes.
7. (Optional) Animations: Use CSS transitions or keyframes for the CTA button or card hover effects.

**Submission**:

1. Push your code to a public GitHub repository.
2. Deploy your site using GitHub Pages, Vercel, or Netlify.
3. Post the link to live website & GitHub repo in this channel. Include a desktop and mobile screenshot of your website.
